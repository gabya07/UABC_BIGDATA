# UABC_BIGDATA
Git Flow Basics

## Basics Commands
```sh
$ git clone <repo_name>
$ git add <-a | .>
$ git commit -m "Comments for description of the main changes"
$ git status
$ git push origin <branch>
```
## Branch Commands
```sh
$ git checkout -b <Branch_name>
$ git add <-a | .>
$ git commit -m "Comments for description of the main changes"
$ git status
$ git push origin <branch>
```
